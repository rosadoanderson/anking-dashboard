Sobre o site

Versão estática pronta para abrir localmente ou publicar no GitHub Pages.

Este site foi gerado a partir do seu arquivo AnKing .apkg e mantém o conteúdo completo em formato de texto para navegação local.

Os dados de desempenho, filas SM-2, tempo, heatmap e meta diária são salvos apenas no navegador onde o site for aberto.

Imagens e áudios do pacote foram substituídos por marcadores como [imagem] e [áudio].

Quantidade importada: cards em decks.
